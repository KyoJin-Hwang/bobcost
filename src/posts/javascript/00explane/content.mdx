## 📘 JavaScript란?

**JavaScript**는 웹 브라우저에서 실행되는 대표적인 프로그래밍 언어로, 웹 페이지를 **동적으로 조작**하고, **사용자와의 상호작용**을 가능하게 합니다.

> ✅ **HTML**은 웹 페이지의 **구조(structure)** 를 만들고,  
> ✅ **CSS**는 그 구조에 **스타일(style)** 을 입히며,  
> ✅ **JavaScript**는 웹 페이지에 **행동(behavior)** 을 부여합니다.

---

### 🧾 정의
- **객체 기반(Object-oriented)**  
- **이벤트 기반(Event-driven)**  
- **동적 타입(Dynamically typed)**  
- **싱글 스레드 기반의 인터프리터 언어(Interpreted language)**
> 브라우저가 HTML과 CSS를 해석한 후, JavaScript는 DOM(Document Object Model)을 제어하여 페이지에 생명력을 불어넣습니다.
---

### 🕘 역사 간단 정리
- **1995년**: Netscape의 브렌던 아이크(Brendan Eich)가 10일 만에 JavaScript의 초안 개발  
- **초기 이름**은 Mocha → LiveScript → JavaScript로 변경  
- Java와는 이름만 비슷할 뿐, 전혀 다른 언어입니다  
- 이후 ECMAScript(표준 스펙)로 발전하여 지속적으로 업그레이드됨
---

### 🧠 JavaScript는 어떤 역할을 하나요?
JavaScript는 웹 페이지에 다음과 같은 **동적 기능**을 제공합니다:
- 버튼 클릭 시 UI 변경  
- 입력값 검증  
- 모달 창 열기/닫기  
- 슬라이더, 드롭다운, 애니메이션 처리  
- 서버와의 데이터 통신 (AJAX, fetch 등)  
- 실시간 채팅, 알림, 비디오 플레이어 제어 등

---

### 🔧 실행 환경

#### 📍 브라우저 (Client-side)
- HTML과 함께 웹 페이지 내에서 실행됩니다.
- 각 브라우저에는 자체 JavaScript 엔진이 존재
  - Chrome → **V8**
  - Firefox → **SpiderMonkey**
  - Safari → **JavaScriptCore**
  - Edge → **Chakra (구버전)**
#### 📍 Node.js (Server-side)
- 2009년 이후, **Node.js** 등장으로 서버 사이드에서도 JavaScript 사용 가능해짐
- 백엔드, CLI 툴, API 서버, 배치 작업 등으로 확장
---

### 🌐 지금의 JavaScript는?
과거에는 브라우저에서만 동작하던 JavaScript가, 지금은 프론트엔드부터 백엔드, 모바일, 데스크탑, IoT, 머신러닝까지 커버하는 범용 프로그래밍 언어로 성장했습니다.


**풀스택 언어**
	- `Frontend` : React, Next.js, Vue.js 등으로 UI 구축
	- `Backend` : Node.js를 통해 서버, API, 배치 시스템 등 구현 가능

**모바일 앱 개발**
	- `React Native`, `Ionic`
  - 하나의 코드베이스로 iOS/Android 앱 개발 가능

**데스크탑 애플리케이션**
	- `Electron`
  - JavaScript + HTML/CSS로 크로스 플랫폼 데스크탑 앱 개발

**IoT (사물인터넷)**
	- `Johnny-Five`, `Espruino`
  - JavaScript로 하드웨어 제어 가능

**머신러닝 / AI**
	- `TensorFlow.js`, `Brain.js`
  - 브라우저에서 직접 딥러닝 모델을 학습하거나 실행 가능
